# Safari Sana Expeditions

Official website for **Safari Sana Expeditions** — a Tanzania-based tour operator offering wildlife safaris, Kilimanjaro climbs, and day trips from Moshi.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the dev server:
   `npm run dev`

## Build & Deploy

Build the production files:
`npm run build`

The `dist/` folder is committed to the repository. The server only needs `git pull` — no build step required on the server.
